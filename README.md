# Chichi's Bizarre Adventure  
### 吉吉的奇妙冒险  
*A 3D Adventure • Puzzle Exploration • Parkour Journey*

---

## About the Game  
**Chichi's Bizarre Adventure** is a 3D adventure and puzzle-exploration game built using Unity URP.  
You play as **Chichi**, a brave, adorable explorer who accepts a mission to repair the broken bridge between the islands.  

But there’s more—  
Chichi soon discovers that his friend, who previously traveled to the island for an expedition, has gone missing for a long time.  
With determination, Chichi embarks on a journey to:

- Fix the bridge
- Explore mysterious areas
- Solve puzzles and perform parkour challenges
- Search for his lost friend

---

## Game Features

### **Core Gameplay**
- Third-person movement (walk, run, jump)
- Parkour elements (platforming & obstacle traversal)
- Puzzle exploration
- Environment interaction  

---

## Controls

| Action | Input |
|--------|--------|
| Move | WASD |
| Run | Shift |
| Jump | Space |
| Attack | Left Mouse |
| Look Around | Mouse movement |
| Zoom Camera | Scroll Wheel |

---

## Game Objectives
1. Explore the island  
2. Collect resources  
3. Break trees and push logs  
4. Repair the bridge  
5. Travel to the next island  
6. Search for your missing friend
